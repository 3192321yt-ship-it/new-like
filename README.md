# 坦克大战 HTML

一个纯 HTML、CSS、JavaScript 实现的 Canvas 坦克大战小游戏。

## 玩法

- `Enter`：开始游戏
- 方向键或 `WASD`：移动坦克
- `Space`：发射炮弹
- `P`：暂停
- `R`：重开

游戏包含敌方坦克 AI、砖墙和钢墙、基地防守、关卡推进、分数和生命系统。直接打开 `index.html` 即可游玩。

## 新增随机内容

- 每一关都会生成不同地图，包含砖墙、钢墙、沙包、水面和伪装林。
- 敌人被击毁后有概率掉落道具，地图中也会定时刷新道具。
- 道具包括护盾、极速、额外生命、冻结敌人、清屏炸弹、基地加固和穿甲星。
- 坦克、地形、子弹和道具使用 Canvas 绘制，增加了阴影、履带、炮塔和材质细节。

## 本地运行

双击 `index.html`，或在项目目录启动一个静态服务器：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。
